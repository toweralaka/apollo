# Apollo
Employee management

- Install Docker
- Run
```console
docker pull node
```
- Run
```console
docker pull mongo
```
- Run

```console
docker run -d -p 27017:27017 --name my-mongo-database mongo
```
- Verify docker is running
```console
docker ps
```
- Run
```console
npm install
```

- Run
```console
npm start
```

- Visit 127.0.0.1:3000
